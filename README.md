# ulauncher-viridian-theme
A simple theme green viridian to ulauncher


## Installation

```sh

mkdir -p ~/.config/ulauncher/user-themes
git clone https://github.com/arthurrio/ulauncher-viridian-theme \
  ~/.config/ulauncher/user-themes/

```
